# 🚲 Yulu Micro-Mobility Demand Analysis

A data science case study focused on understanding the key factors influencing demand for shared electric cycles in India using statistical analysis and machine learning.

---

## 📊 Objective
To identify significant variables affecting bike rental demand and derive actionable business insights using:
- Exploratory Data Analysis (EDA)
- Hypothesis Testing
- Linear Regression Modeling

---

## 🧠 Key Insights
- **Hour of the day** is the strongest predictor (peak commute times drive demand)
- **Temperature** positively impacts demand (optimal range: ~18–28°C)
- **Humidity & bad weather** reduce usage
- **Seasonality** significantly affects demand patterns
- No major difference between **weekday vs weekend** demand

---

## 🧪 Statistical Analysis
- ✅ T-tests (1-sample, 2-sample, paired)
- ✅ Chi-square tests (goodness-of-fit & independence)
- ✅ Normality & variance checks
- ✅ Correlation analysis

---

## 🤖 Model
- Linear Regression model
- **R² Score:** ~35% (moderate predictive power)
- Identified key contributing features to demand

---

## 📁 Dataset
- ~10,000+ records
- Features include:
  - Weather conditions
  - Temperature & humidity
  - Time (hour, day, season)
  - User counts (casual & registered)

---

## 💡 Business Recommendations
- Deploy more bikes during **peak hours**
- Use **weather & temperature forecasts** for dynamic allocation
- Scale operations based on **seasonal demand**
- Maintain **uniform strategy across weekdays & weekends**

---

## 🛠️ Tech Stack
- Python (Pandas, NumPy)
- Visualization: Matplotlib, Seaborn
- Statistics: SciPy
- ML: Scikit-learn

---

## 🚀 Conclusion
This project demonstrates how statistical testing combined with machine learning can drive real-world business decisions in mobility services.
